- All project documentation files must reside under the `docs/` directory in the repository root.
- Whenever a new documentation file is added or an existing one is renamed, it must be registered in the root `README.md` under the "ðŸ“š Documentation" section with a relative link (e.g., `docs/<file>.md`).
