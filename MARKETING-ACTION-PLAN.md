## Фаза 1: Подготовка (Неделя 1)

### Что нужно сделать:

#### 1.1 Настройка Аналитики
- [ ] **Добавить трекинг источников** в бота
  - Реферальные коды для каждого канала
  - UTM-параметры в /start ссылках
  - Пример: `t.me/DoraDuraDoraDuraBot?start=partner_musicchannel`

- [ ] **Создать таблицу метрик** (Google Sheets / Notion)
  ```
  | Дата | DAU | Новые юзеры | Источник | Конверсия в Premium | Доход |
  ```

- [ ] **Настроить логирование событий** в боте:
  - Регистрация (с источником)
  - Первый скачивание
  - Попадание в лимит
  - Апгрейд на Premium/VIP
  - Реферал

#### 1.2 Создание Телеграм-канала
- [ ] Создать канал `@DoraduraNews` или `@DoraduraRu`
- [ ] Оформить: аватар, описание, закреплённый пост
- [ ] Написать 5 постов для старта (заготовки)

#### 1.3 Подготовка Материалов
- [ ] Промо-картинки (3-5 вариантов)
- [ ] Текст для партнёрских постов (3 варианта для A/B)
- [ ] Список 20 потенциальных каналов-партнёров

## Фаза 2: Запуск Гипотез (Недели 2-4)

### Гипотеза 1: Партнёрства с каналами

**Действия:**
```
Неделя 2:
├── Отправить 10 писем админам каналов
├── Договориться о 2-3 постах
└── Бюджет: $60-90

Неделя 3:
├── Опубликовать посты
├── Отслеживать регистрации по коду
└── Замерить CAC
```

**Feedback Loop:**
```
ЗАМЕР (ежедневно):
├── Новые юзеры с кода канала
├── Конверсия в Premium за 7 дней
└── Стоимость за юзера (CAC)

РЕШЕНИЕ (через 7 дней после поста):
├── CAC < $0.50 → Повторить с этим каналом
├── CAC $0.50-1.00 → Попробовать другой креатив
└── CAC > $1.00 → Не продолжать с этим каналом
```

---

### Гипотеза 2: Микроинфлюенсеры

**Действия:**
```
Неделя 3:
├── Найти 10 блогеров (1K-50K подписчиков)
├── Связаться, договориться о цене ($25-50)
└── Подготовить ТЗ для интеграции

Неделя 4:
├── Запустить 3-5 интеграций
├── Уникальный код для каждого
└── Бюджет: $100-150
```

**Feedback Loop:**
```
ЗАМЕР:
├── Просмотры ролика/поста
├── Переходы в бота
├── Регистрации
└── CAC

РЕШЕНИЕ (через 14 дней):
├── ROI > 3x → Масштабировать с этим инфлюенсером
├── ROI 1-3x → Тестировать другой формат
└── ROI < 1x → Не продолжать
```

---

### Гипотеза 3: A/B тесты в боте

**Тест 1: Сообщение при лимите**

**Варианты:**
- A: "Подожди 30 секунд до следующего скачивания"
- B: "Пропустить ожидание? Апгрейд на Premium [Кнопка]"
- C: "Premium юзеры качают без ожидания. Попробуй бесплатно 24ч [Кнопка]"

**Feedback Loop:**
```
ЗАМЕР (на 1000 показов каждого варианта):
├── CTR на кнопку апгрейда
├── Конверсия в Premium за 7 дней
└── Отток (юзер перестал пользоваться)

РЕШЕНИЕ:
├── Вариант с лучшей конверсией и приемлемым оттоком → Внедрить
└── Все варианты хуже контроля → Вернуть старый вариант
```

## Фаза 3: Контент и SEO (Параллельно)

### Действия:
```
Еженедельно:
├── 1-2 поста в Телеграм-канал
├── 1 SEO-статья (Telegraph или лендинг)
└── Анализ позиций в поиске

Темы для статей:
├── "Как скачать музыку с YouTube в Телеграм"
├── "Лучший бот для скачивания видео"
├── "Скачать музыку 320 kbps бесплатно"
└── "Как скачать субтитры с YouTube"
```

**Feedback Loop:**
```
ЗАМЕР (ежемесячно):
├── Позиции в Яндекс/Google
├── Трафик на статьи
├── Переходы в бота
└── Конверсия посетителей в юзеров

РЕШЕНИЕ:
├── Статья в топ-10 → Написать связанные статьи
├── Статья не ранжируется → Переработать или удалить
└── Высокий трафик, низкая конверсия → Улучшить CTA
```

## Фаза 4: Реферальная программа

### Действия:
```
Неделя 2:
├── Улучшить механику рефералов (если нужно)
├── Добавить прогресс-бар к следующей награде
└── Создать сообщение-напоминание о рефералах

Постоянно:
├── Напоминать о рефералах после скачивания
├── Еженедельный топ рефереров в канале
└── "Реферальные выходные" раз в месяц (x2 награда)
```

**Feedback Loop:**
```
ЗАМЕР:
├── % юзеров, отправивших реф-ссылку
├── Конверсия реф-ссылки в регистрацию
├── % новых юзеров от рефералов
└── LTV реферальных юзеров vs органических

РЕШЕНИЕ:
├── Реф-конверсия < 5% → Увеличить награду
├── Мало шерят ссылку → Упростить шеринг
└── Реф-юзеры конвертятся лучше → Больше фокуса на рефералы
```

## Еженедельный Feedback Loop

### Каждый понедельник (30 минут):

```
1. СОБРАТЬ ДАННЫЕ
   ├── Новые юзеры за неделю (по источникам)
   ├── DAU/WAU
   ├── Конверсии (Free → Premium → VIP)
   ├── Доход
   └── CAC по каналам

2. ЗАПОЛНИТЬ ДАШБОРД
   ├── Сравнить с прошлой неделей
   ├── Сравнить с целями
   └── Выделить аномалии

3. ПРИНЯТЬ РЕШЕНИЯ
   ├── Что работает? → Масштабировать
   ├── Что не работает? → Остановить или изменить
   └── Что тестировать на этой неделе?

4. ОБНОВИТЬ ПЛАН
   ├── Скорректировать бюджет
   ├── Добавить новые задачи
   └── Убрать неактуальное
```

## Ежемесячный Feedback Loop

### В конце каждого месяца (2-3 часа):

```
1. ГЛУБОКИЙ АНАЛИЗ
   ├── Когортный анализ (retention по неделям)
   ├── Unit-экономика (LTV, CAC, LTV:CAC)
   ├── Воронка конверсии
   └── Сравнение каналов

2. ПЕРЕСМОТР ГИПОТЕЗ
   ├── Какие гипотезы подтвердились?
   ├── Какие опровергнуты?
   └── Какие новые гипотезы появились?

3. КОРРЕКТИРОВКА СТРАТЕГИИ
   ├── Перераспределение бюджета
   ├── Новые каналы для теста
   └── Изменения в продукте

4. ПЛАНИРОВАНИЕ СЛЕДУЮЩЕГО МЕСЯЦА
   ├── Цели (конкретные числа)
   ├── Ключевые инициативы
   └── Бюджет по каналам
```

## Карта Метрик и Решений

```
┌─────────────────────────────────────────────────────────────────┐
│                    ДЕРЕВО РЕШЕНИЙ                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  DAU падает?                                                    │
│  ├── Да → Проверить retention                                   │
│  │        ├── Retention упал → Проблема с продуктом            │
│  │        └── Retention ок → Проблема с привлечением           │
│  └── Нет → Всё ок, фокус на конверсию                          │
│                                                                 │
│  CAC растёт?                                                    │
│  ├── Да → Проверить по каналам                                  │
│  │        ├── Один канал → Остановить/изменить                 │
│  │        └── Все каналы → Проблема с креативами/оффером       │
│  └── Нет → Можно масштабировать                                │
│                                                                 │
│  Конверсия падает?                                              │
│  ├── Да → Проверить воронку                                     │
│  │        ├── Проблема на этапе X → A/B тест этого этапа       │
│  │        └── Везде падает → Проблема с ценой/ценностью        │
│  └── Нет → Можно привлекать больше трафика                     │
│                                                                 │
│  LTV падает?                                                    │
│  ├── Да → Проверить churn                                       │
│  │        ├── Churn вырос → Работать над retention             │
│  │        └── Churn ок → Проблема с upsell в VIP               │
│  └── Нет → Можно повышать CAC                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Критические Точки Контроля

### Красные флаги (требуют немедленных действий):

| Метрика | Красный флаг | Действие |
|---------|--------------|----------|
| DAU | -30% за неделю | Проверить бота, искать причину |
| CAC | >$1.00 | Остановить платные каналы |
| Конверсия | <0.5% | Пересмотреть ценообразование |
| Retention 7d | <20% | Работать над онбордингом |
| Жалобы | >5% юзеров | Срочно фиксить баги |

### Зелёные сигналы (можно масштабировать):

| Метрика | Зелёный сигнал | Действие |
|---------|----------------|----------|
| CAC | <$0.30 | Увеличить бюджет канала на 50% |
| Конверсия | >5% | Увеличить трафик |
| Retention 7d | >50% | Фокус на привлечение |
| LTV:CAC | >5:1 | Агрессивно масштабировать |
